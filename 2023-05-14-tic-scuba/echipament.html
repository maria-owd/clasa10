Echipament

Echipamentul de scufundare este format dintr-o serie de componente care sunt esențiale pentru o scufundare sigură și confortabilă:

  • butelia de aer comprimat
  • masca
  • vesta BCD - Buoyancy Control Device
  • regulator de aer (detentor)
  • computer de scufundare
  • costumul de neopren
  • labe

Butelia de aer este sursa principală de aer pentru scafandri, iar aceasta trebuie să fie verificată și pregătită înainte de fiecare scufundare.

Regulatorul este folosit pentru a regla presiunea aerului respirat, in functie de presiunea ambientala de la o anumita adancime

Un element important al echipamentului de scufundare este vesta de compensare a flotabilitatii (BCD). Această vestă este utilizată pentru a controla flotabilitatea scafandrului și poate fi umflată sau dezumflată pentru a regla poziția corpului sub apă.

Costumul de neopren oferă protecție împotriva temperaturilor scăzute ale apei și asigură confortul termic. Masca este folosită pentru a vedea sub apă și trebuie să fie adecvată dimensiunilor feței scafandrului, pentru a asigura o etanșeitate adecvată.

Un alt element important al echipamentului de scufundare este computerul de scufundare. Acesta este un dispozitiv electronic care măsoară adâncimea, timpul petrecut sub apă și alte informații importante pentru a ajuta scafandrul să-și gestioneze timpul de scufundare și să evite pericolele cum ar fi decompresia sau lipsa de aer.
